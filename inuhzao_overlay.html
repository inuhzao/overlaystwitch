Viewers em Tempo Real
👁 Viewers Twitch ao vivo
A contagem de viewers é buscada da API pública da Twitch para o canal inuhzao a cada 60 segundos. Funciona automaticamente quando a overlay está aberta e o canal está em live.

Nota: A API pública da Twitch requer autenticação para dados em tempo real. Se o número não aparecer, o canal pode não estar em live neste momento ou pode precisar de um token de API. O número mostrado na overlay é automaticamente atualizado.
Configuração no OBS
1. Guarda este ficheiro no teu computador
Guarda como inuhzao_overlay.html em qualquer pasta.

2. Adiciona Browser Source no OBS
Cenas → clica + em Fontes → Navegador (Browser Source)

3. Usa o URL local do ficheiro
Adiciona ?obs=1 no fim do URL para mostrar só a barra (sem o painel).

4. Dimensões recomendadas
1920Largura (px)
44Altura (px)
1FPS (mínimo)
5. CSS personalizado no OBS
No campo "CSS Personalizado" cola:
body { background: transparent !important; overflow: hidden; }
Dicas
• O relógio atualiza automaticamente a cada minuto
• Os viewers atualizam a cada 60 segundos (quando o canal está em live)
• Para mudar WAGER ↔ RAW: usa o toggle no tab Controlo
• Podes esconder redes sociais individualmente no tab Controlo
• O carrossel alterna automaticamente entre as redes ativas a cada 4s
• Guarda a configuração no tab Controlo — fica guardada no browser
